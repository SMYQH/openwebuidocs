# 👋 Open WebUI 文档

欢迎访问 **Open WebUI** 的官方文档——这是一个自托管、注重隐私且可扩展的 AI 界面，适用于 Ollama 等 LLM 以及与 OpenAI 兼容的 API。

本网站基于 [Docusaurus](https://docusaurus.io/) 构建，包含以下内容：

- 🔧 安装和设置指南（Docker、本地、手动）

- 🧩 插件和扩展文档

- 📚 API 参考和管道使用方法

- 🗂 文件上传和 RAG 集成

- 🤖 开发者贡献指南
