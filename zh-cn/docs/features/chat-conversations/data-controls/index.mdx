Open WebUI 的 **数据控制** 部分提供了用于管理个人数据的工具，包括文件、共享内容和聊天历史。你可以通过 **设置 > 数据控制** 访问它。

## 资源

- [📁 文件管理](./files.md) - 统一管理你上传的所有文档
- [🔗 共享聊天](./shared-chats.md) - 管理并撤销已分享聊天链接的访问权限
- [📦 已归档聊天](./archived-chats.md) - 恢复或永久删除已归档对话
- [💾 导入与导出](./import-export.md) - 备份与恢复聊天历史
