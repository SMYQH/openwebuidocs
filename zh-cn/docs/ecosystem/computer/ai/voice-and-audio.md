# 语音和音频

所有音频相关功能都在**设置 → 音频**中：语音转文字、文字转语音、语音模式和语音备忘录。

## 语音转文字

任何 OpenAI 兼容的转写 API。默认值：基础 URL `https://api.openai.com/v1`，模型 `whisper-1`。一个 STT 配置支持三件事：

- 转写聊天和语音模式中的语音输入
- 语音备忘录的转录文本
- 发送到消息机器人的语音笔记

长录音在转写前会自动分块（此路径需要机器上安装 `ffmpeg`）。

## 文字转语音

默认关闭。任何 OpenAI 兼容的 `/audio/speech` 端点；默认模型 `tts-1`，声音 `alloy`，mp3 输出。回复会**逐句**朗读，因此播放会在回复完成之前开始。播放速度可调，从 0.5 倍到 2 倍。

## 语音模式

免提对话：你说话，AI 回答，回复被读回，麦克风为你的下一轮次重新就绪。STT 源可配置：

- **browser**：浏览器的内置语音识别，无需 API 密钥
- **provider**：上面配置的 STT API

语音模式有自己可自定义的系统提示，与你的聊天提示分开。这对于保持口语回答简短很有用。

## 语音备忘录

从任何设备录制音频，它会作为文件保存到工作区。配置语音转文字后，备忘录会自动转写为 Markdown：从手机上口述审查笔记，然后在办公桌上的编辑器中阅读。
