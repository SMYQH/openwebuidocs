# 安装应用 (PWA)

Open WebUI Computer 是一个 PWA：从浏览器安装它，它会获得一个主屏幕图标，在自己的应用窗口中打开，并出现在你手机的分享表中，这样你就可以直接发送链接和文件到工作区。

## 添加到主屏幕

在手机浏览器中打开你的 Computer URL（使用你的[远程访问设置](./index.md)中的 URL，每次都使用同一个 URL，这样应用始终指向同一个实例）：

- **iPhone/iPad (Safari)：** 分享按钮 → **添加到主屏幕**。
- **Android (Chrome)：** 菜单 → **添加到主屏幕** / **安装应用**。
- **桌面 (Chrome/Edge)：** 地址栏中的安装图标，或菜单 → **安装**。

启动新图标并登录一次。会话会持续，所以从那时起它会直接打开到你的工作区。如果没有出现安装选项，你的浏览器不支持 PWA 安装；在普通标签页中一切仍然可以工作。

## 选择"分享"做什么

在**设置 → PWA** 中，选择如何处理分享的文本和链接：

- **每次都询问**
- **创建聊天草稿**：链接作为消息显示，准备发送给 AI
- **创建笔记文件**：它作为文件保存到工作区

以及分享文件的存放位置：工作区根目录、每次都询问文件夹，或你配置的文件夹。

## 从任何应用分享到你的工作区

在操作系统和浏览器支持 PWA 分享目标的情况下，Open WebUI Computer 会出现在系统分享表中。从任何应用分享 URL、文本或文件，选择 Computer，它会根据你的 PWA 设置到达：文章进入聊天草稿，PDF 进入项目文件夹。在设备没有分享目标支持的情况下，粘贴到新聊天或笔记中，或通过文件浏览器上传。

## 键盘快捷键

使用硬件键盘（平板或桌面）：

| 快捷键 | 操作 |
|---|---|
| `Cmd+K` | 快速搜索（查找任何内容） |
| `Cmd+Shift+F` | 搜索文件内容 |

在 Windows 和 Linux 上使用 `Ctrl` 代替 `Cmd`。
