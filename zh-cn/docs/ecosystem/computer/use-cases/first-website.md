# 在 AI 的陪伴下构建你的第一个网站

你想要一个网站：俱乐部的页面、艺术课的作品集、粉丝页面，都可以。你从未写过 HTML。通常的建议是使用拥有你页面的网站构建器，或一个 YouTube 教程系列。第三种选择：在你自己的电脑上作为真实文件构建，由 AI 编写第一个版本并解释你问到的每一行。

你最终将得到比托管页面更好的东西：你可以阅读、修改、修复和永远保留的实际代码。

**你需要：** 完成[快速入门](/ecosystem/computer/quickstart)并[连接模型](/ecosystem/computer/ai/connect-a-model)。零编程经验。

## 操作步骤

1. **为它创建一个家。** 创建一个新文件夹（你可以在工作区选择器中直接操作），命名为 `my-site`，并作为工作区打开。

2. **描述你想要什么。** 在聊天中：

   > 为我们的机器人俱乐部构建一个单页网站。深色背景、大标题、照片区域和底部的一个笑话。纯 HTML 和 CSS 文件，不要框架。

   批准出现的文件创建。一分钟后，你的文件浏览器中就会有 `index.html` 和 `style.css`。你机器上的真实文件。

3. **实时查看。** 提问：

   > 为此文件夹启动一个本地 Web 服务器，以便我可以预览。

   AI 运行一个小型服务器（如 `python3 -m http.server 8080`；你会看到它要求运行的命令）。端口出现在文件浏览器上方的**端口**行中。点击它，你的站点会在预览标签页中打开。

4. **现在是真正的学习部分。** 在预览旁边的编辑器中打开 `index.html`。更改标题文本，保存，刷新预览。故意破坏一些东西。然后像对导师一样询问 AI：

   > 第 14 行的 `<div>` 是做什么的？为什么照片区域在宽屏上是并排的？

   它读取*你的*文件并解释*你的*代码，这每次都胜过通用教程。

5. **在你的手机上展示。** 如果你已设置[手机访问](/ecosystem/computer/phone-and-remote/)，在手机上打开相同的预览，在真实的移动浏览器中观看你的网站。这就是你的第一个响应式设计课程，完全免费。

## 为什么这样有效

一切都在一个文件夹中作为普通文件：可读、可移植、属于你。AI 完成令人生畏的空白页部分；编辑器与预览并排的循环完成教学；因为聊天存在于工作区中，下周的"添加第二个页面"会直接从你离开的地方继续。当你准备好让世界看到它时，让 AI 带你了解 GitHub Pages 等免费托管服务；文件已经正是你所需要的。
