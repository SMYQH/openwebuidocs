# 文件和编辑器

文件浏览器显示工作区文件夹在磁盘上的实际状态。浏览树，在过滤器中输入以缩小范围，使用面包屑导航返回。文件按类型显示可识别的图标，并附带大小一目了然。

## 处理文件

从浏览器创建文件和文件夹、重命名和删除。从你所在的设备上传，或直接拖放文件。右键点击（触屏长按）文件或文件夹背景以获取上下文菜单，包含常见操作，包括复制文件路径和下载到你浏览的设备。

这里的一切操作在真实文件系统上：浏览器中的重命名就是磁盘上的重命名。

## 编辑

打开一个文件，它出现在编辑器标签页中，带有语法高亮。打开任意数量的文件；将标签页拖入分割区域以并排查看两个文件：实现与其测试并排，配置与其描述的文档并排。分割和打开的标签页是工作区布局的一部分，所以当你回来时会恢复，从任何设备。

## 预览

许多格式会渲染而不是显示原始字节：

| 类型 | 预览 |
| --- | --- |
| Markdown | 渲染文档 |
| 图片 | 内联图片视图 |
| PDF | 页面查看器 |
| CSV / TSV | 表格视图 |
| DOCX / XLSX | 文档和电子表格渲染 |

对于浏览器无法渲染的任何内容，打开原始文本或下载文件。
