# 你的工作区

工作区是运行 Open WebUI Computer 的机器上的一个真实文件夹，而不是上传或副本。将 Computer 指向 `/Users/you/src/project`，你得到的是那个实际目录：它的文件、它的 git 检出、在那里启动的 shell，以及关于它的 AI 聊天。

选择一个工作区将该项目的一切带入一个视图：

- **文件和编辑器**：浏览树，使用语法高亮编辑，预览 Markdown、图片、PDF 和 Office 文档。
- **终端**：主机上的完整 shell，关闭浏览器后继续运行。
- **Git**：分支、差异、暂存、提交、推送/拉取、隐藏和 worktrees，无需命令行。
- **聊天**：限定在项目范围内的 AI 对话，作为文件存储在项目内部。
- **预览和浏览器**：在你的工作旁边打开开发服务器的端口或任何 URL。

## 标签页、分割和布局

一切都在标签页中打开：文件、终端、聊天、浏览器视图、端口预览。拖拽标签页以分割视图，然后再次分割任何部分，任意多次。拖拽分隔线以调整每个部分的大小。

布局按工作区记住，跨设备。在桌面上将终端放在日志旁边、差异放在源代码旁边进行排列，当你从手机或另一台电脑打开该工作区时，相同的安排正在等待你。切换工作区即切换布局；没有任何东西丢失。
