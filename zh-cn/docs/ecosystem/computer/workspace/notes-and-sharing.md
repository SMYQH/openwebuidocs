# 笔记、备忘录和分享

工作区不仅用于代码。笔记、录音和从其他应用发送的内容都作为真实文件进入项目文件夹：可搜索、可版本控制，并且像其他所有内容一样对 AI 可见。

## Markdown 笔记

创建笔记，它是一个 Markdown 文件在工作区中。在编辑器中编辑它，预览渲染效果，随项目一起提交。没有单独的笔记孤岛需要稍后导出。

## 语音备忘录

录制语音备忘录，音频保存在工作区中。配置语音转文字后，录音会自动转写为 Markdown，所以走路时捕捉的想法变成了你可以在办公桌上搜索和编辑的文本文件。在[语音和音频](/ecosystem/computer/ai/voice-and-audio)中设置转写。

## 从其他应用分享

在你的手机上安装 Computer 作为 PWA，它会出现在操作系统分享表中。从任何应用分享链接、文本、图片或 PDF，并选择它变成什么：

- **聊天草稿**：准备好发送给 AI，附带你的问题
- **笔记**：作为文件保存到工作区
- **文件导入**：放入工作区文件夹

在**设置 → PWA** 中设置默认操作和导入目标文件夹。安装应用和手机工作流在[手机应用](/ecosystem/computer/phone-and-remote/phone-app)中有详细介绍。

## 始终有效的备用方案

没有分享表，或在未安装 PWA 的设备上？通过文件浏览器上传：拖放或上传操作将文件直接从任何已登录的浏览器放入工作区。
