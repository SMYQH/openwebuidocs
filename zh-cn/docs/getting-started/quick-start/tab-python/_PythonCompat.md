:::info Python 版本兼容性
Open WebUI 支持 **Python 3.11 和 3.12**。**目前尚不支持 Python 3.13** —— 我们的部分依赖还需要发布兼容 3.13 的版本，在此之前，在 3.13 上安装会失败或在运行时崩溃。

- **生产环境**请使用 **Docker 镜像**或 **最新的 Python 3.11**。这也是我们重点测试的组合。
- **Python 3.12 同样可以工作**，但我们收到过极个别的 3.12 上行为异常的报告，且在 3.11 上未能复现。如果你在 3.12 上遇到难以解释的问题，请先降到最新的 3.11 试试。
:::
