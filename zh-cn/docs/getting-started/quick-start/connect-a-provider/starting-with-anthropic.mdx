## 概览

Open WebUI 原生支持 **Anthropic 的 Claude 模型**，方式是使用其兼容 OpenAI 的端点。只需填入 API 密钥，就可以直接开始聊天——无需中间层、无需 pipe、无需额外配置。

Open WebUI 内置了兼容层，可自动识别 Anthropic URL、获取模型列表并转换请求。聊天补全、流式输出以及**工具调用**都能开箱即用。

:::warning Anthropic 兼容性声明
Anthropic 官方说明其兼容 OpenAI 的 API 主要面向**评估与测试**，而不是生产负载。实际使用中，包括工具调用在内的能力都可以正常工作，但请留意这一官方定位。如果你需要 Claude 原生完整特性（PDF 处理、引用、extended thinking、prompt caching），Anthropic 建议使用其原生 `/v1/messages` API，可通过 [pipe function](/features/extensibility/plugin/functions/pipe) 或 [LiteLLM](https://docs.litellm.ai/) 这类代理接入。
:::

---

## 第 1 步：获取 API 密钥

1. 打开 [console.anthropic.com](https://console.anthropic.com/)。
2. 创建账户或登录。
3. 进入 **API 密钥**，生成新的 API 密钥。

---

## 第 2 步：在 Open WebUI 中添加连接

1. 在浏览器中打开 Open WebUI。
2. 进入 ⚙️ **管理面板** → **连接** → **OpenAI**。
3. 点击 ➕ **添加连接**。
4. 填写以下内容：

| 设置项 | 值 |
|---|---|
| **URL** | `https://api.anthropic.com/v1` |
| **API 密钥** | 你的 Anthropic API 密钥 |

5. 点击 **保存**。

:::tip
在 URL 输入框中键入时，系统会提示常见提供商端点——你只需从下拉列表中选择 Anthropic URL 即可。
:::

---

## 第 3 步：开始聊天！

就是这么简单！Open WebUI 会自动识别 Anthropic 端点并获取可用模型。只需在模型下拉框中选择 Claude 模型，就能开始使用。

流式输出、多轮对话和工具调用都可无缝工作。

:::info 可选：筛选模型
如果你只想显示部分模型，可以在连接设置中的 **模型 ID（筛选）** 白名单里加入指定模型 ID。常见示例：

| 模型 | ID |
|---|---|
| Claude Opus 4.6 | `claude-opus-4-6` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` |
| Claude Haiku 4.5 | `claude-haiku-4-5` |

最新模型 ID 请查看 [Anthropic 模型文档](https://docs.anthropic.com/en/docs/about-claude/models)。
:::

---

## 管理连接

你可以随时通过连接条目旁的开关**启用或禁用** Anthropic 连接。这样你就能临时停用该提供商，而无需删除连接或丢失配置。
