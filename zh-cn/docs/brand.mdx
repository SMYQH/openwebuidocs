## 品牌名称使用

官方名称为 **Open WebUI**。

### 正确用法：
- Open WebUI (在提及品牌或产品时)
- open-webui (通常用于代码仓库或包名称)

### 错误变体 (请勿使用)：

- OpenWeb UI
- OpenWebUI
- OpenWeb
- Open Web UI
- openweb-ui
- OpenWeb Ui
- openWebUI
- Open-Web UI
- open web UI
- open-ui
- open-web-ui

在提及品牌或产品时，请始终将 **Open WebUI** 作为两个独立的单词使用，并如上所示保持正确的大小写。

在提及代码仓库、包或代码库时，请使用小写并带连字符的 **open-webui**。

## 徽标与视觉识别

- 必须直接使用提供的 Open WebUI 徽标，不得进行修改。
- **请勿**拉伸、裁剪或扭曲徽标。
- **请勿**更改徽标颜色或添加特效。
- 在徽标周围留出清晰的空间，以保持其可见性。
- 在干净、不杂乱的背景上使用徽标，以确保其清晰易读。

## 排版

- 使用符合 Web 标准、简单且易读的字体。
- 在所有 Open WebUI 相关材料中保持一致的字体大小和样式。
- 避免使用会降低可读性的装饰性字体。

## 使用规则

- 除非得到明确许可，否则**请勿**以任何暗示背书的方式使用 Open WebUI 的徽标或名称。
- **请勿**将 Open WebUI 品牌整合到你自己的产品或公司品牌中。
- 当 Open WebUI 视觉元素与其他品牌徽标并排使用时，请保持清晰且成比例的间距。
- 始终尊重品牌完整性，并按要求使用提供的资产。

## 联系我们

如有关于品牌资产使用方面的问题，或想申请用于其他用途的许可，请通过官方仓库或项目沟通渠道联系 Open WebUI 团队。
